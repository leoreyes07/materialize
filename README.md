My Blog Website
A responsive static blog website built using HTML5, CSS3, and the Materialize CSS framework.
The project includes a homepage with blog cards, a detailed blog post page, and a newsletter registration page.
________________________________________
Preview
This project features:
•	A homepage with a hero banner and blog post cards
•	Individual blog post page with article content and related posts
•	Newsletter registration page with a sign-up form
•	Responsive layout for desktop, tablet, and mobile devices
________________________________________
Features
•	Responsive design using Materialize CSS
•	Google Material Icons integration
•	Blog post card layout with reveal animations
•	Search input UI for frontend display only
•	Newsletter sign-up form
•	Clean and reusable layout structure
•	Custom CSS styling

Pages Overview
Home Page (index.html)
•	Navigation bar with logo and links
•	Hero banner section
•	Search input field
•	Blog cards with images and reveal content
•	Links to individual blog posts
Blog Post Page (post.html)
•	Full article content layout
•	Sidebar card for additional information
•	Featured image section
•	Related posts displayed as cards
Registration Page (register.html)
•	Newsletter sign-up form
•	Input fields for user details
•	Interest selection using checkboxes
•	Materialize form validation styles
